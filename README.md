# clock

Multiple clocks from different timezones.

# Usage

$ npm install
$ mongodb
$ express
$ start
$ nodemon -dev
$ path

# Uses

HTML 5
CSS 3
ECMAscript 2015
Moment
Moment timezones
MongoDb
Express
node.js
nodemon

npm run watch the index.js file to host it in the localhost:8888.
